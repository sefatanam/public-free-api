# Public APIs without 'api-key'
A collective list of free APIs for use in software and web development.
All credit goes to this [public-apis](https://github.com/public-apis/public-apis) repository.

The purpose of making this repo is jumpstart looking for api without facing extra hassle of sign-up with credit card & many more stuff like this. 
Here about 240 links are vanished which needed credit-card or extra credentials.
